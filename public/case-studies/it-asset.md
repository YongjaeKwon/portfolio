# 교육청 IT 자산관리 솔루션

> 공개용 정리입니다. 내부 코드, 고객사별 상세 정책, 운영 URL, 환경 정보는 제외했습니다.

## 개요

교육청과 학교의 IT 자산을 등록, 배포, 대여, 반납, 폐기, P2P 이동 단계로 관리하는 Spring MVC 기반 업무 시스템입니다. 교육청, 학교, 부서 권한에 따라 조회 범위가 달라지는 점이 핵심입니다.

## 담당 역할

- 자산 등록·대여·반납·폐기 및 P2P 이동/승인 화면 개선
- 교육청 > 학교 > 부서 권한 계층에 따른 조회 범위와 검색 조건 처리
- 대시보드·통계 화면의 상태별 집계와 동적 컬럼 조회 개선
- SMS, 메일, 우편번호, 외부 AS 연동 결과를 화면 흐름에 반영
- MyBatis SQL 조회 조건과 화면 검색 조건 정합성 점검

## 프론트엔드 관점의 핵심

- 권한별로 보이는 데이터 범위가 달라지는 화면에서 조회 조건과 UI 상태를 함께 맞췄습니다.
- 자산 상태가 바뀌는 업무 단계에 따라 사용자가 확인해야 하는 정보와 액션을 정리했습니다.
- 외부 API 결과를 업무 화면의 성공, 실패, 예외 상태로 연결했습니다.

## 사용 기술

Java, Spring MVC, Spring Security, JSP, MyBatis, MariaDB, Tabulator, REST API
