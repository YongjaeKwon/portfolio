# AS / 설치 / 상담 업무관리 시스템

> 공개용 정리입니다. 내부 코드, 고객사별 상세 정책, 운영 URL, 환경 정보는 제외했습니다.

## 개요

AS 접수, 설치, 상담 업무를 데스크톱, 모바일, 태블릿 환경에서 처리하는 WebSquare 기반 운영 시스템입니다. 현장 사용자는 모바일에서 접수와 동의 단계를 진행하고, 내부 담당자는 진행상태 조회와 후속 처리를 확인합니다.

## 담당 역할

- WebSquare / JSP 기반 화면 신규 개발 및 운영 오류 수정
- 모바일 AS 접수, 진행상태 조회, 개인정보 동의, 태블릿 전자서명 화면 구현
- 파일 조회·업로드, 입력값 검증, 단계별 선택 조건 처리
- REST API 응답과 메일·문자·알림톡 발송 결과를 화면 상태로 반영
- 권한과 조회 조건 이슈 재현 및 유사 화면 점검

## 프론트엔드 관점의 핵심

- 모바일과 태블릿에서 같은 업무 데이터가 이어지도록 화면 상태를 정리했습니다.
- 사용자가 접수, 동의, 첨부, 서명, 조회 단계를 진행할 때 필요한 안내와 검증을 화면에 반영했습니다.
- 외부 발송 결과가 성공, 실패, 예외 상태에 따라 다르게 보이도록 처리했습니다.

## 사용 기술

JavaScript, WebSquare, JSP, jQuery, Java, Spring MVC, MyBatis, MariaDB, REST API
