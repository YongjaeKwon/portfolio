# TGS 협력사 포탈 시스템

> 공개용 정리입니다. 내부 코드, 고객사별 상세 정책, 운영 URL, 환경 정보는 제외했습니다.

## 개요

협력사, CE, 교육, 계약, 공지, 설문 데이터를 관리하는 Spring Boot 3 기반 포탈 시스템입니다. 운영자가 반복적으로 처리하는 등록, 업로드, 발송, 조회 업무를 Vue 화면과 Spring Boot API로 연결합니다.

## 담당 역할

- Vue.js 기반 관리자 화면과 Spring Boot API 개발
- 교육 등록, 대상자 엑셀 업로드, 메일 발송, 제출 현황 조회 기능 구현
- 게시판, 제안하기, 설문에서 사용하는 댓글·대댓글 UI와 처리 흐름 공통화
- Google OTP 2단계 인증, IP 기반 예외, 인증 상태별 분기 처리 개선
- 협력사·CE·교육 데이터 조회 조건과 화면 상태 개선

## 프론트엔드 관점의 핵심

- 운영자가 엑셀 업로드 후 대상자 검증, 메일 발송, 제출 현황 확인까지 이어서 처리할 수 있도록 화면 흐름을 구성했습니다.
- Tabulator 그리드와 Vue 상태를 함께 다루며 행 선택, 조회 조건, 결과 표시를 맞췄습니다.
- 인증 상태와 권한 조건에 따라 사용자가 볼 수 있는 화면과 액션을 분기했습니다.

## 사용 기술

Vue.js, JavaScript, Tabulator, Chart.js, Java, Spring Boot, Spring Security, MyBatis, MariaDB, Oracle
